# Codeinfra Schema

This package defines [Codeinfra
Schema](https://www.codeinfra.com/docs/guides/codeinfra-packages/schema/), a
language-neutral specification for cloud resource models. Codeinfra Schema is the
interface definition language for all Codeinfra Packages, and is used as the input
to SDK code generation for each supported Codeinfra language.
